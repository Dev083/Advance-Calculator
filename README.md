# Advanced Calculator

Welcome to the Advanced Calculator, a Python-based calculator that can perform various mathematical operations ranging from basic arithmetic to advanced trigonometric functions.

## Installation

To use the calculator, follow these steps:

1. Clone the repository to your local machine.
2. Install the required packages by running `pip install -r requirements.txt`.
3. Run the `main.py` file using a Python interpreter.

## Usage

The calculator can be utilized through the command-line interface (CLI). Here are some examples of how to use it:

- To add two numbers:
  ```
  python main.py add 5 3
  ```

- To subtract two numbers:
  ```
  python main.py subtract 5 3
  ```

- To multiply two numbers:
  ```
  python main.py multiply 5 3
  ```

- To divide two numbers:
  ```
  python main.py divide 5 3
  ```

- To calculate the exponentiation of two numbers:
  ```
  python main.py exponentiate 5 3
  ```

- To calculate the modulus of two numbers:
  ```
  python main.py modulus 5 3
  ```

- To calculate the square root of a number:
  ```
  python main.py sqrt 5
  ```

- To calculate the sine of an angle:
  ```
  python main.py sin 30
  ```

- To calculate the cosine of an angle:
  ```
  python main.py cos 30
  ```

- To calculate the tangent of an angle:
  ```
  python main.py tan 30
  ```

## Features

- Basic mathematical operations such as addition, subtraction, multiplication, and division.
- Exponentiation.
- Modulus.
- Square root.
- Trigonometric functions such as sine, cosine, and tangent.
- Error handling for invalid inputs.
